Introduction

A client-first sustainability hub that estimates annual CO₂e, gamifies habits, and offers AI coaching.

CarbonWise Coach is a React 19 single-page application with an optional Express backend for Gemini AI. User profiles, challenge progress, and emission history persist entirely in the browser via localStorage — no server-side user database.

How to read this doc Use the sidebar to jump to Flows for visual diagrams, or Architecture for file-level detail. Markdown version: ARCHITECTURE.md.

Features

Four emission sectors, tabbed dashboard, simulator, achievements, PDF export, and Smart Coach.

Emission sectors

🚗

Transportation

Commute distance, vehicle type, flights — calculated via carbonCalculator.ts

🥗

Diet & Food

Meat frequency, sourcing, food waste parameters

⚡

Home Utilities

Home size, appliances, clean energy tariff choices

🛍️

Shopping & Retail

Garment updates, device lifecycle, recycling habits

Additional capabilities

  • Tabbed dashboard — Overview, Take Action, Simulator, Progress (reduces scroll fatigue)
  • What-if sandbox — Sliders for transport, diet, energy before applying to profile
  • Weekly challenges — Mon–Sun streak tracker with emission impact
  • Achievements — Badges unlocked from footprint and habit milestones
  • Historical trends — Recharts area chart of logged footprints
  • PDF reports — Client-side jsPDF journey summary
  • Smart Coach — Gemini chat + insights (rules-engine fallback offline)
  • Accessibility — Skip link, modal focus traps, ARIA tabs, jsx-a11y lint

API & Smart Coach Flow

Gemini runs server-side only. Production Netlify static deploy uses rules-engine fallback.

Smart Coach request (Node / local dev)
SmartCoach.tsx
→
POST /api/coach-chat
or /api/coach-insights
→
Zod validate body
Rate limit 60/15min
→
Gemini API
(GEMINI_API_KEY)
→
Zod validate response
→
Render in UI
Netlify static deploy caveat Express /api/* routes do not run on plain Netlify. Smart Coach automatically falls back to recommendations.ts rules engine when API calls fail. To enable live AI on Netlify, add Netlify Functions or host the Node server separately.

Security

  • API keys server-side only
  • Helmet + CSP in production
  • Rate limit: 60 req / 15 min on AI endpoints
  • Zod validation on all POST bodies and AI responses
  • Chat truncated to 20 messages (client + server)
  • Request body max 32 KB
  • No server-side user database — localStorage only
